# My Ecommerce App

A small demo ecommerce app built with React, Vite, and TypeScript.

## Features
- Product list page  
- Product details page  
- Search bar  
- Tailwind CSS + MUI UI  
- Artificial loading delay  
- Offline cache using IndexedDB (localForage)  

## Tech Stack
- React 18  
- Vite  
- TypeScript  
- Tailwind CSS  
- Material UI  
- localForage  
- React Router  

## Run Locally
```bash
npm install
npm run dev
```
